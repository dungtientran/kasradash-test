## Getting Started

Clone the repo:

```
git clone https://github.com/dungtientran/kasradash-test.git
```

```
cd kasradash-test
```

```
npm install
```

```
npm run dev
```

## Overview

- Framework - **Nextjs15 app router**
- Language - **TypeScript**
- Styling - **Tailwind CSS**
- Components - **Shadcn-ui**
- Schema Validations - **Zod**
- State Management - **Zustand**
